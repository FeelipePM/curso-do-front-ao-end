# Dicas do Terminal
## Listando comandos aprendido na aula:

- `cd` (navega entre as pastas)

**Exemplo para entrar dentro da pasta:**
```
cd nomeDaPasta
```
<hr>

- `cd ..` (volta para a pasta anterior)

**Exemplo para sair de uma pasta:**
```
cd ..
```

<hr>

- `mkdir` (cria pastas)

**Exemplo como criar uma pasta:**

```
mkdir nomeDaPasta
```
